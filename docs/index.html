    Authentication

    Pake JSON Web Token, yang didapet pas login. Jadi, nanti harus ada header ini kalau mau ngakses endpoint yang ada authenticationnya:

    Authorization: jwt_yg_didapet_dari_login

    User

    Login

    Request Header

    Content-type: application/json

    Response jika request sukses

    Token dipakai di next request yang butuh Authorization header.

    {
    "message": "logged in",
    "token": "your_jwt_token"
}

    Response jika request gagal

    {
    "error_message": "error_message",
    "error_code": 500 // tergantung errornya
}

    Digunakan untuk login biar dapet token.

    HTTP Request

    POST /login

    Request Parameter

    Parameter Type Keterangan
    email string -
    password string -
    fcm_token string fcm token didapat setelah insialisasi fcm di hp

    HTTP Status Code

    Code Keterangan
    200 Login Sukses
    400 Bad Request (Tidak ada request data)
    401 Tidak bisa login (unauthorized)
    500 Server Error

    Register

    Request Header

    Content-type: application/json

    Response jika request sukses

    {
    "message": "user registered"
}

    Response jika request gagal

    {
    "error_message": "error_message",
    "error_code": 500 // tergantung errornya
}

    Digunakan untuk user dapat register ke sistem.

    HTTP Request

    POST /register

    Request Parameter

    Parameter Type Keterangan
    username string -
    email string -
    password string -
    full_name string -
    gender string "female" atau "male"
    fcm_token string fcm token didapat setelah insialisasi fcm di hp

    HTTP Status Code

    Code Keterangan
    200 Login Sukses
    400 Bad Request (Tidak ada request data)
    500 Server Error

    Promo

    New Promo

    Request Header

    Authorization: your_jwt_token
Content-type: application/json

    Response jika request sukses

    {
    "message": "add promo success"
}

    Response jika request gagal

    {
    "error_message": "error_message",
    "error_code": 500 // tergantung errornya
}

    Digunakan untuk membuat promo dari pengguna.

    HTTP Request

    POST /promo/new

    Request Parameter

    Parameter Type Keterangan
    name string -
    address string -
    latitude double latitude posisi pengguna
    longitude double longitude posisi pengguna
    description string -
    start_date string -
    end_date string -
    allowed_gender string value "male", "female" atau "both"
    max_slot int jumlah maksimal yg dapat join promo

    HTTP Status Code

    Code Keterangan
    200 Login Sukses
    400 Bad Request (Tidak ada request data)
    500 Server Error
